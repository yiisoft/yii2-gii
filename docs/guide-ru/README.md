Расширение Gii для Yii 2
========================

Gii - это веб-приложение для генерирования кода в приложениях Yii2.
Gii позволяет Вам быстро сгенерировать модели, формы, модули, CRUD, и т.п.

Gii предоставляет веб-интерфейс для интерактивного создания кода. Это расширение также
предоставляет инструмент командной строки для тех, кто в основном предпочитает работать в консоли.
